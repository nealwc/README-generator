# README Generator

## Description

README Generator is a time saving command-line application that creates a README.md from users input. 

The README will be generated based of the following prompts answered by the user: 

- Project title
- Description
- Table of Contents
- Installation
- Usage
- License
- Contributing
- Tests
- Questions
- User GitHub email
